Study-Hub

A realtime Pomodoro and sessiontracking web app with live leaderboards. Originally built in March2025, publicly deployed now (August2025).

Tech Stack

Frontend: React (Vite) 路 Tailwind CSS 路 React Router 路 Axios

Backend: AWS Lambda (Serverless Framework) 路 APIGateway 路 DynamoDB

Local Dev: serverless-offline 路 AWS CLI

Hosting: Vercel (Free tier)

Features

Group Collaboration: Create or join study groups in real time.

Session Tracking: Start/pause/stop timers and log focused study sessions.

Pomodoro Mode: Configurable focus/break cycles with automatic transitions.

Live Leaderboard: See whos studied the most minutes each week.
